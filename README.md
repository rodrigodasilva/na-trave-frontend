## Getting started

1. Clone the project.

   ```bash
   git clone git@github.com:rodrigodasilva/blip.git
   ```

2. Access the project directory.

   ```bash
   cd blip
   ```   

3. Install dependencies.

   ```bash
   npm install
   ```

6. Start dev server with hot reload at http://localhost:3000.
   ```bash
   npm run dev
   ```

## Other commands

### Lint commands

```bash
npm run lint
```

### Build commands

```bash
npm run build
```

### Run the app in production mode at http://localhost:3000.

```bash
npm run serve
```

### Test commands

- Run unit tests and watch
  ```bash
  npm run test:unit
  ```
- Run unit tests with coverage
  ```bash
  npm run test:unit:coverage
  ```
