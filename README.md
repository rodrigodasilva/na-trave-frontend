## Getting started

1. Clone the project.

   ```bash
   git clone git@github.com:rodrigodasilva/na-trave-frontend.git
   ```

2. Access the project directory.

   ```bash
   cd na-trave-frontend
   ```   

3. Install dependencies.

   ```bash
   npm install
   ```

6. Start dev server with hot reload at http://localhost:3000.
   ```bash
   npm run dev
   ```
